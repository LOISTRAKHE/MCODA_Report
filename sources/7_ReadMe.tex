\singlespacing
The project cantains a folder of dataset, and 2 python code named baseline optimization and pareto.

The baseline optimization.py runs 9 algorithms (Ping-pong, Greedy by Size, Hole-aware Greedy, Local Search, SA, GA based on buffers and Local Search, SA, GA based on offsets) and generates 2 csv data.

All results detail notes down the feasiblility, peak memory, fragmentation and execution time of every dataset using every algorithm.

All results summary calculates the mean data of 4 different stuctured networks using 9 different algorithms.

The baseline optimization.py also generates memory allocation plots for every dataset using every algorithms in the folder named polts.

Parameter optimization tool is blocked in the baseline optimization.py, one can turn on it by delete \# before $tune_local_search(SA,GA)$.

After running the baseline optimization.py, pareto plots can be generated by running pareto.py.
